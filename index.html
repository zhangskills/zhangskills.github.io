使用hexo搭建静态博客

发表于 2015-05-09   |  

Hexo 是一个简单地、轻量地、基于Node的一个静态博客框架。通过Hexo我们可以快速创建自己的博客，然后部署在github pages搭建的免费无限流量的个人博客。

官方文档

Hexo：http://hexo.io/docs/

GitHub Pageshttps://help.github.com/articles/what-are-github-pages/

hexo使用

依赖

  1. Node.js 官网：http://nodejs.org/
  2. Git 官网：http://git-scm.com/

安装

npm install -g hexo-cli

创建博客

hexo init blog

会在当前目录下创建一个名为“blog”的文件夹，

安装依赖

cd blog

npm install

进入博客目录，由于hexo采用node.js开发，安装所有依赖

创建文章

hexo new 文章标题

创建一个标题为“文章标题”的博客

编辑文章

直接打开博客目录下source/_posts/文章标题.md文件进行编辑，比如：

本地启动

hexo server

启动hexo服务器，通过http://localhost:4000/访问

查看帮助

hexo 或者 hexo help

部署到github

  1. github上创建一个“你的用户名.github.io”的项目，例：yourname.github.io.git

  2. 修改博客目录下全局配置文件：_config.yml，找到deploy或添加：

  1. 执行部署命令：hexo deploy

  2. 用浏览器打开http://yourname.github.io 访问博客

域名绑定

  1. 域名解析中添加一条cname记录指向yourname.github.io

  2. 在博客目录source文件夹下创建一个名为“CNAME”的文件，内容为你的域名，重新部署博客

官方文档：https://help.github.com/articles/setting-up-a-custom-domain-with-github-pages/
